How to create low poly graphics 1 Hour

Starting to learn photoshop pen tool with organized structures.

Photoshop pen tool

Photoshop provides multiple Pen tools. The standard Pen tool draws with the greatest precision; the Freeform Pen tool draws paths as if you were drawing with pencil on paper, and the magnetic pen option lets you draw a path that snaps to the edges of defined areas in your image.

View Setting

Under View menu

Go Show > chose Grid

Use pen tool to select a square area then do " Command + enter "

" Command + enter " is shortcut to make selection..

1. Use pen tool (shortcut: p) to select a shape on grid

2. Command + enter

3. Go Filter menu > Blur > Average (fill in an aberage color) (shortcut: command + f)

Then continue to repeat steps(1,2,3) again and again
